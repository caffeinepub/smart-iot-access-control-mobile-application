# Specification

## Summary
**Goal:** Update the app's color scheme across all six themes to use a fresh, cohesive palette with emerald green or teal as the primary accent color instead of blue/purple.

**Planned changes:**
- Update CSS custom properties for all six themes (dark, light, amber, cyberpunk, midnight, ocean) with a new color palette
- Replace the primary accent color with emerald green or teal, applied consistently to buttons, badges, active nav links, status indicators, charts, and highlights
- Update Tailwind config color tokens to match the new palette
- Ensure sufficient contrast ratios for text readability in both light and dark modes

**User-visible outcome:** The app displays a refreshed visual theme with emerald green or teal as the dominant accent color across all theme variants, replacing previous blue/purple tones.
